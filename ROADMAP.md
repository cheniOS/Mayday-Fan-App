# 开发路线图 (Mayday Fan App)

## 阶段一：核心验证 (Week 1)
- [ ] 使用 SwiftUI 创建项目原型。
- [ ] 跑通 MusicKit 权限请求与当前播放歌曲（Song ID）抓取。
- [ ] 实现 MapKit 基础展示。

## 阶段二：云端联动 (Week 2)
- [ ] 配置 CloudKit Container。
- [ ] 实现“上传坐标 + 歌曲 ID”到公共数据库。
- [ ] 实现地图实时渲染其他在线粉丝的“星海点”。

## 阶段三：视觉与交互 (Week 3)
- [ ] 设计毛玻璃质感的 UI 卡片。
- [ ] 开发“点亮荧光棒”轻互动功能。
- [ ] 集成演唱会行程管理列表。

## 阶段四：合规与商业化 (Week 4)
- [ ] 集成 Apple In-App Purchase (IAP) 会员逻辑。
- [ ] 寻找亲属身份注册个体户或代办备案。
- [ ] 准备 App Store 提交物料（截图、描述）。
